# Chrome 扩展完整使用指南

## 🎯 为什么选择浏览器扩展？

相比传统的爬虫方式，Chrome 扩展有以下优势：

1. **✅ 避免反爬虫**: 在浏览器环境中运行，就像正常用户访问
2. **✅ 无需 Token**: 直接读取页面内容，不需要获取 API token
3. **✅ 实时提取**: 可以实时看到提取的内容
4. **✅ 用户友好**: 图形界面，操作简单
5. **✅ 跨平台**: 支持 Windows、macOS、Linux

## 📋 功能清单

### 已实现功能

- ✅ 单篇文章下载
- ✅ Markdown 格式导出
- ✅ 图片下载（可选）
- ✅ 元数据提取（标题、作者、时间等）
- ✅ 下载历史记录
- ✅ 美观的用户界面

### 计划功能

- 🔄 批量下载优化
- 🔄 导出为 PDF
- 🔄 导出为 HTML
- 🔄 自定义下载路径
- 🔄 文章分类管理

## 🚀 快速开始

### 1. 安装扩展

参考 [INSTALL.md](INSTALL.md) 进行安装。

### 2. 使用扩展

1. **打开一篇微信公众号文章**
   ```
   例如: https://mp.weixin.qq.com/s/xxxxx
   ```

2. **点击扩展图标**
   - 在浏览器工具栏找到扩展图标
   - 点击打开弹出窗口

3. **下载文章**
   - 点击"下载当前文章"按钮
   - 文件会自动下载

## 📖 详细使用说明

### 单篇下载

**适用场景**: 下载当前正在阅读的文章

**步骤**:
1. 在微信公众号文章页面
2. 点击扩展图标
3. 点击"下载当前文章"
4. 等待下载完成

**输出文件**:
- 文件名: `文章标题.md`
- 位置: Chrome 默认下载文件夹
- 格式: Markdown，包含元数据和正文

### 批量下载

**适用场景**: 下载公众号的历史文章

**步骤**:
1. 打开公众号的历史文章列表页面
2. 打开扩展
3. 设置下载数量（1-100）
4. 点击"开始批量下载"
5. 按照提示操作

**注意**: 
- 由于微信的限制，批量下载可能需要手动翻页
- 建议每次下载 10-20 篇，避免被限制

### 设置选项

#### 下载图片

- **开启**: 会下载文章中的所有图片到单独文件夹
- **关闭**: 只下载 Markdown 文件，图片链接保留原 URL

#### 包含元数据

- **开启**: Markdown 文件开头包含文章信息（标题、作者、时间等）
- **关闭**: 只包含文章正文

## 🔧 高级功能

### 自定义下载路径

目前文件会下载到 Chrome 默认下载文件夹。要更改：

1. 打开 Chrome 设置
2. 找到"下载内容"
3. 更改下载位置

### 批量下载技巧

1. **分批下载**: 不要一次下载太多，建议每次 10-20 篇
2. **间隔时间**: 每下载几篇后稍作等待
3. **手动翻页**: 在历史文章页面手动翻页，然后下载

### 处理特殊内容

- **视频**: 目前不支持视频下载，只保留链接
- **音频**: 目前不支持音频下载，只保留链接
- **表格**: 会转换为 Markdown 表格格式
- **代码块**: 会保留代码格式

## 🐛 故障排除

### 问题 1: 扩展图标不显示

**原因**: 可能缺少图标文件

**解决**:
1. 检查 `icons/` 文件夹是否存在
2. 运行 `python create_icons.py` 创建图标
3. 或参考 `icons/README.md` 手动创建

### 问题 2: 点击下载没有反应

**检查清单**:
1. ✅ 是否在微信公众号文章页面？
2. ✅ 页面是否完全加载？
3. ✅ 浏览器控制台是否有错误？（按 F12 查看）
4. ✅ 扩展是否已启用？

**解决**:
1. 刷新页面
2. 重新加载扩展
3. 检查浏览器控制台错误

### 问题 3: 下载的文件是空的

**原因**: 可能页面结构变化或内容被限制

**解决**:
1. 等待页面完全加载
2. 检查页面是否可以正常访问
3. 尝试其他文章测试

### 问题 4: 图片下载失败

**原因**: 
- 图片可能有防盗链
- 网络问题
- 图片链接失效

**解决**:
1. 检查网络连接
2. 可以手动保存图片
3. 图片链接会保留在 Markdown 中

## 💡 使用技巧

### 技巧 1: 快速下载多篇文章

1. 打开多篇文章在不同标签页
2. 依次切换到每篇文章
3. 点击扩展图标下载
4. 重复操作

### 技巧 2: 批量处理

1. 下载多篇文章
2. 使用 Markdown 编辑器批量处理
3. 或编写脚本处理下载的文件

### 技巧 3: 保存设置

扩展会记住你的设置（下载图片、包含元数据等），下次使用时会自动应用。
