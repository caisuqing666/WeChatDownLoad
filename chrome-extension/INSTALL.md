# 安装指南

## 快速安装（3步）

### 1. 准备文件

确保 `chrome-extension` 文件夹包含以下文件：
- ✅ manifest.json
- ✅ popup.html
- ✅ popup.css
- ✅ popup.js
- ✅ content.js
- ✅ background.js
- ✅ icons/ 文件夹（可选，有图标更好）

### 2. 打开 Chrome 扩展管理

在 Chrome 浏览器中：
- 输入地址：`chrome://extensions/`
- 或：菜单 → 更多工具 → 扩展程序

### 3. 加载扩展

1. 打开右上角的"开发者模式"开关
2. 点击"加载已解压的扩展程序"
3. 选择 `chrome-extension` 文件夹
4. 完成！

## 详细步骤（带截图说明）

### 步骤 1: 打开扩展管理页面

```
方法1: 在地址栏输入 chrome://extensions/
方法2: 菜单 → 更多工具 → 扩展程序
方法3: 快捷键 (Mac: Cmd+Shift+X, Windows: Ctrl+Shift+X)
```

### 步骤 2: 启用开发者模式

在扩展管理页面右上角，找到"开发者模式"开关，点击打开。

### 步骤 3: 加载扩展

1. 点击"加载已解压的扩展程序"按钮
2. 在弹出的文件选择器中，找到并选择 `chrome-extension` 文件夹
3. 点击"选择文件夹"或"打开"

### 步骤 4: 验证安装

安装成功后，你会看到：
- 扩展出现在扩展列表中
- 浏览器工具栏出现扩展图标
- 没有错误提示

## 图标文件说明

如果看到"缺少图标"的警告，可以：

### 选项 1: 创建简单图标（推荐）

创建以下尺寸的 PNG 图片：
- `icons/icon16.png` - 16x16 像素
- `icons/icon48.png` - 48x48 像素  
- `icons/icon128.png` - 128x128 像素

可以使用任何图片编辑工具，或在线工具：
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/

### 选项 2: 使用占位符

创建一个简单的图标：
```bash
# 在 chrome-extension 目录下创建 icons 文件夹
mkdir -p icons

# 可以使用 ImageMagick 创建简单图标（如果已安装）
convert -size 128x128 xc:#667eea -pointsize 60 -fill white -gravity center -annotate +0+0 "📰" icons/icon128.png
```

### 选项 3: 暂时忽略

没有图标不影响功能，只是显示默认图标。

## 验证安装

安装完成后：

1. **打开一篇微信公众号文章**
   - 例如：https://mp.weixin.qq.com/s/xxx

2. **点击浏览器工具栏的扩展图标**

3. **应该看到弹出窗口**
   - 显示"微信公众号文章下载器"
   - 有"下载当前文章"按钮

4. **测试下载**
   - 点击"下载当前文章"
   - 应该能成功下载 Markdown 文件

## 故障排除

### 问题 1: 找不到"加载已解压的扩展程序"按钮

**解决**: 确保已打开"开发者模式"开关

### 问题 2: 提示"清单文件无效"

**解决**: 
1. 检查 manifest.json 文件是否存在
2. 检查 JSON 格式是否正确（可以使用 JSON 验证工具）
3. 确保所有必需的文件都存在

### 问题 3: 扩展加载后没有图标

**解决**:
1. 检查 icons 文件夹是否存在
2. 检查图标文件是否存在
3. 可以暂时忽略，功能不受影响

### 问题 4: 点击扩展图标没有反应

**解决**:
1. 检查是否在微信公众号文章页面
2. 刷新页面后重试
3. 查看浏览器控制台是否有错误（F12）

### 问题 5: 下载功能不工作

**解决**:
1. 检查 Chrome 下载权限是否被阻止
2. 检查下载文件夹是否有写入权限
3. 查看浏览器控制台错误信息

## 更新扩展

如果修改了代码：

1. 在扩展管理页面找到本扩展
2. 点击刷新图标 🔄
3. 或重新加载扩展文件夹

## 卸载扩展

1. 在扩展管理页面找到本扩展
2. 点击"移除"按钮
3. 确认删除
