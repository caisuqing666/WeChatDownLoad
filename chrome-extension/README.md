# 微信公众号文章下载器 - Chrome 扩展版

一个在浏览器中直接运行的微信公众号文章下载工具，无需担心反爬虫机制。

## ✨ 特性

- 🚀 **浏览器端运行**: 直接在页面中提取内容，避免 IP 封禁
- 📝 **Markdown 导出**: 自动转换为 Markdown 格式
- 🖼️ **图片下载**: 支持下载文章中的图片
- 📚 **批量下载**: 支持批量下载公众号历史文章
- 💾 **下载历史**: 记录下载历史，方便管理
- 🎨 **美观界面**: 现代化的用户界面

## 📦 安装步骤

### 方法一：开发者模式安装（推荐）

1. **下载扩展文件**
   - 确保 `chrome-extension` 文件夹完整

2. **打开 Chrome 扩展管理页面**
   - 在地址栏输入：`chrome://extensions/`
   - 或通过菜单：更多工具 → 扩展程序

3. **启用开发者模式**
   - 在页面右上角打开"开发者模式"开关

4. **加载扩展**
   - 点击"加载已解压的扩展程序"
   - 选择 `chrome-extension` 文件夹
   - 点击"选择文件夹"

5. **完成安装**
   - 扩展图标会出现在浏览器工具栏
   - 点击图标即可使用

### 方法二：打包安装

1. **打包扩展**
   ```bash
   # 在 chrome-extension 目录下
   zip -r wechat-downloader.zip . -x "*.git*" "*.DS_Store"
   ```

2. **安装打包文件**
   - 在扩展管理页面点击"加载已解压的扩展程序"
   - 或使用 Chrome 开发者模式加载

## 🎯 使用方法

### 下载单篇文章

1. 打开任意一篇微信公众号文章
2. 点击浏览器工具栏中的扩展图标
3. 点击"下载当前文章"按钮
4. 文件会自动下载到默认下载文件夹

### 批量下载

1. 在公众号历史文章页面
2. 打开扩展，设置下载数量
3. 点击"开始批量下载"
4. 按照提示操作（由于微信限制，可能需要手动翻页）

### 设置选项

- **下载图片**: 是否下载文章中的图片
- **包含元数据**: 是否在 Markdown 中包含作者、时间等信息

## 🖼️ 图标制作

如果缺少图标文件，可以：

1. **使用在线工具生成**
   - 访问 https://www.favicon-generator.org/
   - 上传或设计图标
   - 下载不同尺寸的图标

2. **使用简单占位符**
   - 创建 16x16、48x48、128x128 的 PNG 图片
   - 可以使用纯色背景 + 文字

3. **临时方案**
   - 扩展在没有图标时会显示默认图标
   - 功能不受影响

## 🔧 开发说明

### 修改代码后重新加载

1. 在扩展管理页面找到本扩展
2. 点击刷新图标 🔄
3. 或重新加载扩展文件夹

### 调试

- **Popup 调试**: 右键扩展图标 → 检查弹出内容
- **Content Script 调试**: 在文章页面按 F12，在 Console 中查看
- **Background 调试**: 在扩展管理页面点击"service worker"链接

## ⚠️ 注意事项

1. **首次使用**: 可能需要刷新文章页面才能正常工作
2. **权限提示**: Chrome 可能会提示权限请求，请允许
3. **下载位置**: 文件会下载到 Chrome 默认下载文件夹
4. **批量下载**: 由于微信限制，批量下载可能需要手动操作

## 🐛 常见问题

### Q: 点击下载没有反应？

A: 
1. 确保在微信公众号文章页面（URL 包含 `mp.weixin.qq.com/s/`）
2. 刷新页面后重试
3. 检查浏览器控制台是否有错误

### Q: 无法提取文章内容？

A:
1. 等待页面完全加载
2. 检查页面是否被微信限制访问
3. 尝试刷新页面

### Q: 图片下载失败？

A:
1. 某些图片可能有防盗链保护
2. 可以手动保存图片
3. 检查网络连接
