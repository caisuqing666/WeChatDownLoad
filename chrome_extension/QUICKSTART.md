# 快速开始 - 5分钟上手

## 第一步：准备图标（1分钟）

### 最简单的方法

1. 打开 `create_icons.html` 文件（在浏览器中）
2. 点击"生成图标"按钮
3. 右键每个图标，选择"图片另存为"
4. 保存到 `icons/` 文件夹，命名为：
   - `icon16.png`
   - `icon48.png`
   - `icon128.png`

### 或者使用 Python 脚本

```bash
pip install Pillow
python create_placeholder_icons.py
```

## 第二步：安装扩展（1分钟）

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 `chrome_extension` 文件夹
6. ✅ 完成！

## 第三步：使用扩展（1分钟）

1. 访问任意一篇微信公众号文章
   - 例如：`https://mp.weixin.qq.com/s/xxxxx`
2. 点击浏览器工具栏中的扩展图标 📥
3. 在弹窗中查看文章信息
4. 选择格式（Markdown/HTML/文本）
5. 点击"下载文章"按钮
6. 文件自动下载到默认下载文件夹

## 完成！🎉

现在您可以：
- ✅ 在浏览器中直接下载文章
- ✅ 无需配置，无需 token
- ✅ 支持多种格式导出
- ✅ 自动下载图片
