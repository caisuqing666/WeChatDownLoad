# 批量下载故障排除指南

## 当前问题：文章列表为空

从控制台日志看到：`提取到的文章列表: ▸ Array(0)`

## 可能的原因

### 1. 公众号主页需要登录
- **症状**：页面显示需要登录或验证
- **解决**：确保已登录微信，在微信中打开公众号主页

### 2. 页面结构变化
- **症状**：无法找到文章链接
- **解决**：扩展已更新提取逻辑，请重新加载扩展

### 3. 页面未完全加载
- **症状**：页面还在加载中
- **解决**：等待页面完全加载后再点击"获取文章列表"

## 调试步骤

### 步骤 1: 检查页面状态

1. 打开公众号主页（`mp.weixin.qq.com/mp/profile_ext?action=home&__biz=...`）
2. 按 F12 打开开发者工具
3. 在 Console 中输入：
   ```javascript
   // 检查是否有文章链接
   document.querySelectorAll('a[href*="/s/"]').length
   ```
4. 如果返回 0，说明页面中没有文章链接

### 步骤 2: 检查是否需要登录

1. 查看页面内容
2. 如果显示"请登录"或需要验证，说明需要登录
3. **解决方法**：
   - 在微信中打开公众号主页
   - 或使用微信网页版登录

### 步骤 3: 手动滚动加载

1. 在公众号主页手动滚动到底部
2. 等待加载更多文章
3. 重复几次，确保加载了足够多的文章
4. 然后再点击"获取文章列表"

### 步骤 4: 使用控制台调试

在扩展弹窗的控制台中（右键扩展图标 → 检查弹出内容），查看：
- `提取到的文章列表` 的详细信息
- 是否有错误信息
- 选择器找到了多少个元素

## 临时解决方案

### 方案 1: 从单篇文章页面获取

1. 打开任意一篇公众号文章
2. 点击扩展图标
3. 切换到"批量下载"模式
4. 点击"获取该作者的所有文章"
5. 扩展会自动跳转到公众号主页并提取文章

### 方案 2: 手动添加文章

如果自动获取失败，可以：
1. 手动打开每篇文章
2. 使用"单篇下载"功能
3. 逐个下载

### 方案 3: 使用 Python 脚本

如果浏览器扩展无法工作，可以使用项目中的 Python 脚本版本。

## 常见问题

### Q: 为什么在单篇文章页面无法获取列表？

A: 单篇文章页面不包含其他文章链接。扩展会：
1. 从当前文章提取公众号ID
2. 自动跳转到公众号主页
3. 从主页提取文章列表

### Q: 为什么提取到的列表是空的？

A: 可能原因：
1. 公众号主页需要登录
2. 页面结构发生变化
3. 公众号没有公开文章
4. 页面未完全加载

### Q: 如何确认页面是否加载完成？

A: 检查：
1. 页面不再显示"加载中"
2. 可以看到文章列表
3. 滚动到底部可以加载更多
