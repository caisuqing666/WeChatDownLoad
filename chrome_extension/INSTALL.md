# Chrome 扩展安装指南

## 快速安装（5分钟）

### 步骤 1: 准备图标

扩展需要图标文件。有两种方式：

**方式一：使用提供的生成器（推荐）**

1. 在浏览器中打开 `create_icons.html`
2. 点击"生成图标"按钮
3. 右键每个图标，选择"图片另存为"
4. 保存到 `chrome_extension/icons/` 文件夹
5. 重命名为 `icon16.png`, `icon48.png`, `icon128.png`

**方式二：使用占位图标**

如果暂时没有图标，可以创建简单的占位图片：

```bash
# 在 chrome_extension 目录下创建 icons 文件夹
mkdir -p chrome_extension/icons

# 可以使用任何图片编辑工具创建 16x16, 48x48, 128x128 的 PNG 图片
# 或者使用在线工具：https://www.favicon-generator.org/
```

### 步骤 2: 加载扩展

1. **打开 Chrome 扩展管理页面**
   - 在地址栏输入：`chrome://extensions/`
   - 或：菜单 → 更多工具 → 扩展程序

2. **开启开发者模式**
   - 在页面右上角，开启"开发者模式"开关

3. **加载扩展**
   - 点击"加载已解压的扩展程序"按钮
   - 选择 `chrome_extension` 文件夹
   - 点击"选择文件夹"

4. **确认安装**
   - 扩展应该出现在扩展列表中
   - 浏览器工具栏应该显示扩展图标

### 步骤 3: 测试使用

1. 访问任意一篇微信公众号文章，例如：
   ```
   https://mp.weixin.qq.com/s/xxxxx
   ```

2. 点击浏览器工具栏中的扩展图标

3. 应该看到弹窗显示文章信息

4. 选择格式并点击下载按钮

5. 文件应该自动下载

## 故障排除

### 问题：扩展无法加载

**可能原因：**
- 缺少图标文件
- manifest.json 语法错误
- 文件路径不正确

**解决方法：**
1. 检查 `icons/` 文件夹中是否有三个图标文件
2. 检查 `chrome://extensions/` 页面中的错误信息
3. 确保所有文件都在 `chrome_extension` 文件夹中

### 问题：扩展图标不显示

**解决方法：**
1. 确保图标文件存在且命名正确
2. 图标必须是 PNG 格式
3. 重新加载扩展

### 问题：无法检测到文章

**解决方法：**
1. 确保在微信公众号文章页面（URL 包含 `mp.weixin.qq.com/s/`）
2. 刷新页面
3. 等待页面完全加载后再点击扩展图标
4. 检查浏览器控制台是否有错误

### 问题：下载失败

**解决方法：**
1. 检查浏览器下载权限设置
2. 确保有足够的磁盘空间
3. 检查文件名是否包含非法字符
4. 尝试使用不同的导出格式

## 更新扩展

当修改了扩展代码后：

1. 打开 `chrome://extensions/`
2. 找到"微信公众号文章下载器"
3. 点击刷新图标（🔄）
4. 重新测试功能

## 卸载扩展

1. 打开 `chrome://extensions/`
2. 找到"微信公众号文章下载器"
3. 点击"移除"按钮
4. 确认删除

## 开发者模式调试

### 查看 Popup 日志

1. 右键扩展图标
2. 选择"检查弹出内容"
3. 在开发者工具中查看 Console

### 查看 Content Script 日志

1. 打开微信公众号文章页面
2. 按 F12 打开开发者工具
3. 在 Console 标签中查看日志

### 查看 Background 日志

1. 打开 `chrome://extensions/`
2. 找到扩展
3. 点击"service worker"链接
4. 在开发者工具中查看日志

## 文件权限说明

扩展需要以下权限：

- **activeTab**: 访问当前标签页内容
- **downloads**: 下载文件到本地
- **storage**: 保存用户设置（可选）
- **scripting**: 注入脚本到页面

这些权限都是必要的，扩展不会收集或上传任何数据。
