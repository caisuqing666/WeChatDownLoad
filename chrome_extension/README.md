# 微信公众号文章下载器 - Chrome 扩展

一个安全、便捷的 Chrome 浏览器扩展，用于在浏览器中直接下载微信公众号文章。

## 为什么选择浏览器扩展？

1. **更安全**: 在真实浏览器环境中运行，不会被封 IP
2. **无需 token**: 直接访问页面 DOM，无需手动获取 token
3. **实时提取**: 在页面加载时自动提取内容
4. **用户友好**: 点击按钮即可下载，无需命令行

## 功能特性

- ✅ 自动检测微信公众号文章页面
- ✅ 提取文章标题、作者、发布时间
- ✅ 支持 Markdown、HTML、纯文本格式导出
- ✅ 自动下载文章中的图片
- ✅ 美观的用户界面
- ✅ 无需配置，开箱即用

## 安装步骤

### 1. 准备图标文件

扩展需要图标文件，请创建以下图标（或使用占位符）：

- `icons/icon16.png` (16x16 像素)
- `icons/icon48.png` (48x48 像素)
- `icons/icon128.png` (128x128 像素)

**快速创建图标的方法：**

1. 使用在线工具生成：https://www.favicon-generator.org/
2. 或使用简单的占位图片（见下方脚本）

### 2. 加载扩展

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 `chrome_extension` 文件夹
6. 扩展安装完成！

### 3. 使用扩展

1. 访问任意一篇微信公众号文章
2. 点击浏览器工具栏中的扩展图标
3. 在弹窗中查看文章信息
4. 选择导出格式（Markdown/HTML/文本）
5. 点击"下载文章"按钮
6. 文件会自动下载到默认下载文件夹

## 使用说明

### 下载单篇文章

1. 打开微信公众号文章页面
2. 点击扩展图标
3. 确认文章信息正确
4. 选择格式并点击下载

### 下载文章（含图片）

1. 点击"下载文章（含图片）"按钮
2. 文章和图片会分别下载
3. 图片保存在 `文章标题_images/` 文件夹中

### 支持的格式

- **Markdown (.md)**: 适合编辑和阅读
- **HTML (.html)**: 保持原始格式
- **纯文本 (.txt)**: 仅文本内容

## 技术实现

### Content Script

在微信公众号文章页面中运行，负责：
- 提取文章标题、作者、发布时间
- 提取文章正文内容
- 提取所有图片 URL
- 提取公众号 ID 和 token（如果可用）

### Popup Script

扩展弹窗的逻辑，负责：
- 与 content script 通信获取数据
- 格式转换（HTML → Markdown/Text）
- 触发文件下载
- 用户界面交互

### Background Service Worker

后台服务，负责：
- 处理扩展安装事件
- 消息路由
- 后台任务处理

## 常见问题

### Q: 扩展无法检测到文章？

A: 确保：
1. 在微信公众号文章页面（URL 包含 `mp.weixin.qq.com/s/`）
2. 页面已完全加载
3. 刷新页面后重试

### Q: 下载的文件是空的？

A: 可能是内容提取失败，尝试：
1. 刷新页面
2. 等待页面完全加载
3. 重新点击扩展图标

### Q: 图片下载失败？

A: 图片下载可能因为：
1. 网络问题
2. 图片链接失效
3. 跨域限制

可以手动保存图片或使用其他工具下载。

### Q: 如何更新扩展？

A: 在 `chrome://extensions/` 页面：
1. 找到扩展
2. 点击刷新按钮
3. 或重新加载扩展文件夹

## 开发说明

### 修改代码后重新加载

1. 在 `chrome://extensions/` 页面
2. 找到扩展
3. 点击刷新图标
4. 重新打开文章页面测试

### 调试

- **Popup**: 右键扩展图标 → 检查弹出内容
- **Content Script**: 在文章页面按 F12，在 Console 中查看
- **Background**: 在 `chrome://extensions/` 中点击"service worker"链接

## 安全说明

- 不会收集或上传任何数据
- 所有处理都在本地完成
- 需要下载权限以保存文件
