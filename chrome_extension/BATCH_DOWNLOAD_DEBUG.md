# 批量下载调试指南

## 快速诊断

### 步骤 1: 检查扩展是否正确加载

1. 打开 `chrome://extensions/`
2. 找到"微信公众号文章下载器"
3. 确保已启用
4. 检查是否有错误提示

### 步骤 2: 测试单篇下载

1. 打开任意一篇微信公众号文章
2. 点击扩展图标
3. 切换到"单篇下载"模式
4. 尝试下载一篇文章
5. 如果单篇下载正常，说明基础功能正常

### 步骤 3: 测试批量下载

1. 打开一篇公众号文章
2. 点击扩展图标
3. 切换到"批量下载"模式
4. 点击"获取该作者的所有文章"
5. 观察状态提示

## 常见错误及解决方法

### 错误 1: "未找到文章列表"

**原因：**
- 无法从当前页面提取文章链接
- 公众号主页需要登录
- 页面结构发生变化

**解决方法：**
1. 确保在微信公众号页面
2. 尝试访问公众号主页：
   ```
   https://mp.weixin.qq.com/mp/profile_ext?action=home&__biz=公众号ID
   ```
3. 手动滚动页面加载更多文章
4. 刷新页面后重试

### 错误 2: "无法获取公众号信息"

**原因：**
- URL 中不包含 `__biz` 参数
- 页面未完全加载

**解决方法：**
1. 刷新文章页面
2. 等待页面完全加载
3. 从文章页面获取（而不是从主页）

### 错误 3: 批量下载卡住

**原因：**
- 某篇文章加载超时
- 网络问题
- 内容提取失败

**解决方法：**
1. 点击"取消"停止下载
2. 减少批量下载数量
3. 检查网络连接
4. 查看控制台错误

### 错误 4: 下载的文件为空

**原因：**
- 文章内容提取失败
- 页面结构变化

**解决方法：**
1. 刷新文章页面
2. 等待页面完全加载
3. 检查控制台是否有错误
4. 尝试单篇下载测试

## 如果仍然无法工作

请提供以下信息：

1. **浏览器版本** - Chrome 版本号
2. **扩展版本** - manifest.json 中的版本
3. **错误信息** - 控制台中的完整错误
4. **操作步骤** - 详细的操作步骤
5. **页面URL** - 当前页面的URL（隐藏敏感信息）

## 临时解决方案

如果批量下载仍然无法工作，可以：

1. **手动批量下载**：
   - 打开每篇文章
   - 使用单篇下载功能
   - 逐个下载

2. **减少批量数量**：
   - 每次选择5-10篇文章
   - 分批下载

3. **使用 Python 脚本**：
   - 使用项目中的 Python 脚本版本
   - 需要手动获取 token
