# 调试指南

## 如何查看扩展程序的错误信息

### 方法 1：检查 Popup 控制台（推荐）

1. 打开扩展程序弹窗（点击扩展图标）
2. **右键点击弹窗内的任意位置**
3. 选择"检查"或"Inspect"
4. 在弹出的开发者工具中，点击"Console"（控制台）标签
5. 查看是否有红色错误信息

### 方法 2：检查扩展程序页面

1. 打开 `chrome://extensions/`
2. 找到"微信公众号文章下载器"
3. 点击"service worker"链接（如果有）
4. 查看控制台错误

### 方法 3：检查 Content Script

1. 打开微信公众号文章页面
2. 按 F12 打开开发者工具
3. 在 Console 标签中查看错误

## 常见问题排查

### 问题：点击下载按钮没有反应

**可能原因：**
- 扩展程序未正确加载
- 文章内容未正确提取
- JavaScript 错误

**解决方法：**
1. 检查状态栏显示的信息
2. 查看控制台错误
3. 刷新文章页面后重试

### 问题：PDF 下载失败

**可能原因：**
- PDF 库未加载
- 图片加载失败
- 内存不足

**解决方法：**
1. 重新加载扩展程序
2. 尝试其他格式（HTML、Markdown）是否正常
3. 如果其他格式正常，说明问题在 PDF 生成
4. 查看控制台的具体错误信息

### 问题：其他格式可以下载，但 PDF 不行

**解决方法：**
- 使用备用方案：PDF 生成失败时会自动使用打印功能
- 在打印对话框中选择"另存为 PDF"

## 测试步骤

1. **测试基本功能：**
   - 打开一篇微信公众号文章
   - 点击扩展图标
   - 检查是否显示文章信息

2. **测试 Markdown 下载：**
   - 选择 Markdown 格式
   - 点击下载
   - 检查是否成功

3. **测试 HTML 下载：**
   - 选择 HTML 格式
   - 点击下载
   - 检查是否成功

4. **测试 PDF 下载：**
   - 选择 PDF 格式
   - 点击下载
   - 观察状态栏信息
   - 查看控制台错误（如果有）

## 报告问题

如果问题仍然存在，请提供：
1. 状态栏显示的具体错误信息
2. 控制台中的错误信息（截图或复制文本）
3. 使用的格式（Markdown/HTML/PDF）
4. 浏览器版本
