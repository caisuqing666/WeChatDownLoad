# 微信公众号下载工具 - 小白使用指南

## 🎯 最简单的使用方法

### 方法一：一键运行（推荐）

我已经为您创建了一个自动测试脚本，直接运行即可：

```bash
cd ~/Documents/GitHub/wechatDownload
python3 test_download.py
```

这个脚本会自动：
- 使用您微信中的文章URL
- 下载文章内容
- 保存到 `downloads/` 文件夹

### 方法二：交互式使用（更灵活）

1. **打开终端（Terminal）**
   - 按 `Cmd + Space` 搜索 "Terminal"
   - 或：应用程序 → 实用工具 → 终端

2. **进入项目目录**
   ```bash
   cd ~/Documents/GitHub/wechatDownload
   ```

3. **运行交互式脚本**
   ```bash
   python3 example_usage.py
   ```

4. **按照提示操作**
   - 输入 `1` 选择"下载单篇文章"
   - 粘贴文章URL（从微信中复制）
   - token 直接按 `Enter` 跳过（不需要）

## 📝 详细步骤

### 步骤 1: 获取文章URL

1. 在微信中打开任意一篇公众号文章
2. 点击右上角的 `...` 菜单
3. 选择"复制链接"
4. 或者直接复制浏览器地址栏的URL

### 步骤 2: 运行下载工具

**最简单的方式**（我已经帮您准备好了）：

```bash
cd ~/Documents/GitHub/wechatDownload
python3 test_download.py
```

**或者使用交互式方式**：

```bash
cd ~/Documents/GitHub/wechatDownload
python3 example_usage.py
```

然后：
- 选择 `1`（单篇下载）
- 粘贴文章URL
- token 按 `Enter` 跳过

### 步骤 3: 查看下载的文件

下载完成后，文件保存在：
```
~/Documents/GitHub/wechatDownload/downloads/
```

**在 Finder 中查看**：
```bash
open downloads/
```

## ❓ 常见问题

### Q: 我需要在终端输入什么？

A: 只需要输入：
```bash
cd ~/Documents/GitHub/wechatDownload
python3 test_download.py
```

### Q: 如何复制粘贴命令？

A: 
1. 选中命令文本
2. 按 `Cmd + C` 复制
3. 在终端中按 `Cmd + V` 粘贴
4. 按 `Enter` 执行

### Q: 下载的文件在哪里？

A: 在 `downloads/` 文件夹中，可以用以下命令打开：
```bash
open downloads/
```

### Q: 需要登录或注册吗？

A: 不需要！单篇下载完全免费，不需要任何账号。

### Q: 可以批量下载吗？

A: 可以，但批量下载需要 token。建议先用单篇下载测试。

## 💡 提示

- **第一次使用**：建议先用 `test_download.py` 测试
- **日常使用**：使用 `example_usage.py` 更灵活
- **查看文件**：用 `open downloads/` 在 Finder 中打开
