# 微信公众号文章下载并合并为PDF工具 - 使用说明

## 🎯 功能特点

- ✅ **单篇/多篇下载**：无需token，直接提供文章链接即可
- ✅ **批量下载**：需要token，可下载历史所有文章
- ✅ **PDF合并**：多篇文章自动合并为一个PDF文件
- ✅ **进度显示**：实时显示下载进度

## 📦 安装依赖

首次使用需要安装PDF相关依赖：

```bash
cd ~/Documents/GitHub/wechatDownload
pip3 install -r requirements.txt
```

或者单独安装：

```bash
pip3 install markdown weasyprint PyPDF2
```

## 🚀 使用方法

### 方法一：运行PDF下载工具（推荐）

```bash
cd ~/Documents/GitHub/wechatDownload
python3 wechat_pdf_downloader.py
```

### 操作步骤

1. **选择模式**
   - 输入 `1`：单篇/多篇下载（无需token）
   - 输入 `2`：批量下载（需要token）

2. **输入文章链接**（模式1）
   - 每行输入一个文章URL
   - 输入空行结束
   - 示例：
     ```
     https://mp.weixin.qq.com/s/第一篇文章
     https://mp.weixin.qq.com/s/第二篇文章
     https://mp.weixin.qq.com/s/第三篇文章
     (按回车,输入空行结束)
     ```

3. **输入公众号名称**
   - 用于PDF封面显示
   - 可选，直接回车跳过

4. **输入PDF文件名**
   - 不含扩展名
   - 直接回车使用默认名称

5. **等待完成**
   - 程序会自动下载所有文章
   - 显示进度：`[1/3]`, `[2/3]`, `[3/3]`
   - 每篇文章显示：`正在下载...` → `√ 文章标题`

## 📂 输出文件

- **Markdown文件**：每篇文章单独保存为 `.md` 文件
- **PDF文件**：每篇文章单独保存为 `.pdf` 文件
- **合并PDF**：所有文章合并为一个 `.pdf` 文件

文件保存在：`downloads/` 文件夹

## ⚠️ 注意事项

1. **PDF功能需要额外依赖**
   - 如果未安装依赖，只会下载Markdown格式
   - 安装命令：`pip3 install markdown weasyprint PyPDF2`

2. **批量下载需要token**
   - 模式2需要从微信浏览器中获取token
   - 建议先用模式1测试（无需token）

3. **网络问题**
   - 如果下载失败，可能是网络问题或文章已删除
   - 程序会自动跳过失败的文章，继续下载其他文章

## 🔧 故障排除

### 问题：PDF功能不可用

**解决**：
```bash
pip3 install markdown weasyprint PyPDF2
```

### 问题：下载失败

**可能原因**：
- 文章URL无效或已过期
- 网络连接问题
- 微信公众号的反爬虫机制

**解决**：
- 检查URL是否正确
- 检查网络连接
- 稍后重试

### 问题：PDF合并失败

**解决**：
- 检查是否安装了 `PyPDF2`
- 单个PDF文件仍然会保存，可以手动合并

## 💡 提示

- **首次使用**：建议先用模式1测试单篇文章
- **多篇文章**：可以一次输入多篇文章链接
- **查看文件**：下载完成后，运行 `open downloads/` 查看文件
