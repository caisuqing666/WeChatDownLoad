# 微信公众号下载工具 - 小白使用指南

## 🎯 最简单的使用方法

### 方法一：直接运行测试脚本（推荐）

我已经为您创建了一个自动测试脚本，只需要在终端输入一行命令：

```bash
cd ~/Documents/GitHub/wechatDownload
python3 test_download.py
```

**就这么简单！** 脚本会自动：
1. 使用您微信中的文章URL
2. 下载文章内容
3. 保存到 `downloads/` 文件夹

### 方法二：下载指定文章

如果您想下载其他文章，可以修改 `test_download.py` 文件中的URL，或者使用交互式脚本：

```bash
cd ~/Documents/GitHub/wechatDownload
python3 example_usage.py
```

然后：
1. 输入 `1`（选择单篇下载）
2. 粘贴文章URL
3. 按 `Enter`（跳过token，免token下载）

## 📝 详细步骤

### 第一步：打开终端

1. 按 `Command + Space`（空格键）
2. 输入 "终端" 或 "Terminal"
3. 按 `Enter` 打开

### 第二步：进入项目文件夹

在终端中输入：

```bash
cd ~/Documents/GitHub/wechatDownload
```

按 `Enter` 执行

### 第三步：运行下载脚本

输入：

```bash
python3 test_download.py
```

按 `Enter` 执行

### 第四步：查看下载的文件

下载完成后，文件会保存在 `downloads/` 文件夹中。

在 Finder 中打开：

```bash
open downloads/
```

或者在终端查看：

```bash
ls downloads/
```

## ❓ 常见问题

### Q: 提示 "command not found: python3"

**解决**：尝试使用 `python` 代替 `python3`：

```bash
python test_download.py
```

### Q: 提示 "No module named 'requests'"

**解决**：安装依赖：

```bash
pip3 install -r requirements.txt
```

### Q: 下载失败怎么办？

**可能原因**：
1. 网络连接问题
2. 文章URL无效或已过期
3. 微信公众号的反爬虫机制

**解决方法**：
- 检查网络连接
- 确认文章URL是否正确
- 尝试使用 Chrome 扩展版本（更稳定）

### Q: 文件保存在哪里？

**位置**：`~/Documents/GitHub/wechatDownload/downloads/`

在 Finder 中打开：
```bash
open ~/Documents/GitHub/wechatDownload/downloads/
```

## 💡 小贴士

1. **不需要 token**：单篇下载完全免费，不需要任何配置
2. **支持格式**：下载的文件是 Markdown 格式（.md），可以用任何文本编辑器打开
3. **批量下载**：如果需要批量下载，可以使用 Chrome 扩展版本（更简单）
